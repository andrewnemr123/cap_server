HoverBot with ESP32
====================

The HoverBot runs on a ESP32-WROVER-E development board. Read [../ESP32-WROVER-E_Datasheet.pdf](../ESP32-WROVER-E_Datasheet.pdf) for pinouts and other specifications about the board.

Please check [ESP-IDF docs](https://docs.espressif.com/projects/esp-idf/en/latest/get-started/index.html) for getting started instructions on ESP-IDF projects.

For more detailed documentation on the ESP32 boards, read [../ESP32_Technical_Reference.pdf](../ESP32_Technical_Reference.pdf).
